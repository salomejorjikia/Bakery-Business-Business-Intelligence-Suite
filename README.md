 🥐 Bakery Business Intelligence Suite

An interactive end-to-end Power BI analytics suite designed to optimize sales strategies, evaluate product performance, and deliver predictive future growth insights for a multi-branch bakery enterprise tracking over $1M in revenue.

## 🎯 Business Problem & Objective
Bakery management was struggling to evaluate isolated branch performance, track actual profit margins against a catalog of expanding product lines, and forecast seasonal inventory demands. 

The objective of this project was to synthesize historical transaction data into an executive-facing analytics model to identify revenue drivers, call out underperforming categories, and provide a 5-year predictive framework.

## 🛠️ Data Architecture & Tech Stack
* **Tool:** Power BI Desktop
* **Data Source:** Microsoft SQL Server
* **Ingestion Method:** Imported clean relational tables into Power BI Desktop via native SQL database connectors.
* **Database Optimization:** Wrote optimized SQL queries using `INNER JOIN`s, filtered clauses, and data-type casting to pre-aggregate complex transactional schemas before loading them into the BI layer. This significantly reduced data load overhead.
* **DAX Engineering:** Developed robust calculated measures to isolate Key Performance Indicators (KPIs) like revenue growth tracking and dynamic margins.

## 🔮 Strategic Business Recommendations
<img width="2075" height="1200" alt="project_page-0005" src="https://github.com/user-attachments/assets/43203c7f-9ecf-481c-9054-9c4b387162db" />

Based on the dashboard's analytics, the following actions were delivered in the final report module:
1. **Portfolio Optimization:** Initiate a portfolio review for the *Biscuits* category due to flat sales volume and poor margin yield.
2. **Expansion Tracking:** Expand inventory pipelines for top-tier future earners flagged by the predictive engine (*Apple Pie, Eclair, and Cottage Cheese Balls*).
3. **Operational Alignment:** Maintain balanced resource sharing between Store 1 and Store 2, as visual mapping confirms revenue is split evenly (~49% to ~51%) between both locations.
